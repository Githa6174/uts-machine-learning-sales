# Video Game Global Sales Prediction 🎮

Proyek ini adalah sistem prediksi numerik berbasis *Machine Learning* menggunakan Python untuk memprediksi total penjualan global video game. Prediksi dilakukan berdasarkan skor kritikus (*Critic Score*), skor pengguna (*User Score*), dan umur rilis game. Proyek ini disusun sebagai bagian dari Ujian Tengah Semester (UTS) mata kuliah Pembelajaran Mesin.

## 🛠️ Teknologi yang Digunakan
- **Bahasa:** Python 3.8+
- **Library:** NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn
- **Environment:** Jupyter Notebook

## 🗂️ Struktur Repository
- `UTS_Prediksi_Penjualan.ipynb` : *Source code* utama (Jupyter Notebook).
- `Video_Games_Sales.csv` : Dataset historis penjualan video game (Kaggle).
- `requirements.txt` : Daftar library Python yang dibutuhkan.
- `Laporan_UTS_Githa.pdf` : Laporan lengkap analisis EDA, evaluasi model, dan kesimpulan.
